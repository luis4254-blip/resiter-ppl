🎯 ¿Cómo usar esta aplicación?

Esta aplicación te permite gestionar las hojas de seguridad de productos químicos y determinar si pueden almacenarse juntos de forma segura.

1️⃣ Lista de Productos

Muestra todos los productos con su información clave y enlaces directos a las hojas SDS en la nube.

2️⃣ Niveles de Riesgo

BAJO Productos de bajo riesgo

MEDIO Requieren cuidado

ALTO Manejo especializado

3️⃣ Código de Colores

✅ Compatible - Almacenamiento permitido

❌ Incompatible - No almacenar juntos

⚠️ Precaución - Evaluar condiciones

⚠️ ADVERTENCIA DE SEGURIDAD

Esta tabla es una guía general. Siempre consulte las hojas de seguridad específicas (SDS) de cada producto antes de almacenarlos juntos. En caso de duda, asuma incompatibilidad y almacene por separado.
